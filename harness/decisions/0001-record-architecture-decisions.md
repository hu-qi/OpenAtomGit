# ADR 0001: Record Architecture Decisions

## Status

Accepted

## Context

OpenAtomGit will be developed with human and AI collaboration. Important design choices need to be recorded.

## Decision

Use ADR files under `harness/decisions/` to record major architecture choices.

## Consequences

- Every major direction change has context.
- Future AI coding can read decisions before editing code.
- Small implementation changes do not require ADR.
