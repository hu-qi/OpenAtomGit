# Manual Test: Auth

- Open app.
- Tap start button.
- Login page appears.
- Tap login button with empty token.
- Error message appears.
- Enter invalid token.
- Tap login button.
- Loading text appears.
- Token validation failed message appears.
- Enter valid AtomGit token.
- Tap login button.
- Main page appears.
- Open Account tab.
- Current user nickname and username appear.
- Token saved locally appears.
- Tap Logout.
- Login page appears again.
