# Native Bridges

This file records HarmonyOS capabilities that may be used by OpenAtomGit.

## Planned capabilities

- HTTP request: AtomGit API access
- Preferences: token and settings
- RelationalStore: repository, issue and history cache
- WebView: OAuth and Markdown rendering
- Clipboard: copy token, link and raw file content
- Share: share repository or issue link

## MVP scope

MVP only needs HTTP request and preferences.
